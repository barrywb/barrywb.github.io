Java 多线程和线程同步

发表于 2022-03-30

进程和线程

  • 进程和线程

    • 操作系统中运⾏多个软件

    • ⼀个运⾏中的软件可能包含多个进程

    • ⼀个运⾏中的进程可能包含多个线程

  • CPU 线程和操作系统线程

    • CPU 线程
      • 多核 CPU 的每个核各⾃独⽴运⾏，因此每个核⼀个线程
      • 「四核⼋线程」：CPU 硬件⽅在硬件级别对 CPU 进⾏了⼀核多线程的⽀持（本质上依然是每个核⼀个线程）
    • 操作系统线程：操作系统利⽤时间分⽚的⽅式，把 CPU 的运⾏拆分给多条运⾏逻辑，即为操作系统的线程
    • 单核 CPU 也可以运⾏多线程操作系统

  • 线程是什么：按代码顺序执⾏下来，执⾏完毕就结束的⼀条线

    • UI 线程为什么不会结束？因为它在初始化完毕后会执⾏死循环，循环的内容是刷新界⾯
